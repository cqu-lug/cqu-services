# CQU Services
CQU Services是一个CQU校园信息聚合仓库。

# 如何提交到CQU Services
- 对于LUG外部人员，fork后再提交PR
- 对于LUG内部人员，使用创建新分支的方式提交PR

# 对部分信息有异议？
去ISSUE吐个痛快！

# 想加入LUG？
去我们官网踩踩吧！
